# starfield
Classic 2D starfield effect using SFML and OpenGL shaders.

## Building
```
mkdir build
cd build
cmake ../ -DCMAKE_BUILD_TYPE=Release
make
```

## Usage
```
./starfield
```
Fullscreen
```
./starfield -f
```
